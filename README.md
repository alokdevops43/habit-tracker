# 📝 Habit Tracker

A clean and responsive **Habit Tracker** built using **HTML, CSS, and JavaScript**. It helps users build positive habits by tracking daily completion, monitoring streaks, and visualizing progress—all with data stored locally in the browser.

---

##Live link

https://alokdevops43.github.io/habit-tracker/

## ✨ Features

* ➕ Add new habits
* ✅ Mark habits as completed
* 🔥 Daily streak counter
* 📊 Progress bar
* 📈 Dashboard statistics
* 🗑️ Delete habits
* 🌙 Dark/Light mode
* 💾 LocalStorage support
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* LocalStorage API
